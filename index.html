Clases de Git and GitHub

git init

Crea un repositorio git

git status

muestra una lista de los archivos que se encuentran en el stage

  • git status -s -b

    muestra una lista de los archivos pero sin comentarios, en una sola linea, y con -b muestra la rama en la que se encuentran

git add [arvhivo en especifico]

Agrega los archivos alstage

  • git add "*.txt"

    Agrega todos los txt de TODO el proyecto

  • git add *.txt

    Agrega todos los txt en el directorio actual

  • git add --all

    Agrega todos los archivos

  • git add < lista de archivos >

    Agrega los archivos que listemos

  • git add pdfs/.pdf

    Agrega todos los PDFs dentro de la carpeta PDFs

  • git add pdfs/

    Agrega todos los archivos dentro de la carpeta PDFs

git reset [arvhivo en especifico]

Quita los archivos que se encuentren en el stage

git commit -m "[agregamos el mensaje que queremos enviar]“

Todo lo que se encuentra en el stage pasa al repositorio

  • git commit -am "[agregamos el mensaje]"

    Se agrega al stage y se agrega al repositorio

  • git commit --amend -m "[agregamos el mensaje]"

    Se corrije el mensaje del commit que está apuntando

git log

Muestra una lista de todos los commits realizados

  • git log --oneline --decorate --all --graph

    Se muestra los commits alineados, decorados, todos y de forma grafica

git reflog

Muestra un historial de cambios que se realizaron

Ramas

git branch

Muestra la lista de todas las ramas

  • git branch ["Nombre de la rama"]

    Crea la rama con ese nombre

git checkout ["Nombre de la rama"]

Te mueve a la rama con ese nombre

git diff ["Nombre de la rama"] ["Nombre de la otra rama"]

Hace una comparación de las ramas, mostrando que fue lo que se cambió
